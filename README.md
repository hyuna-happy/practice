SQL
=
```
관계형 데이터베이스 관리 시스템(RDBMS)의 데이터를 관리하기 위해 만들어진 프로그래밍 언어
```
데이터 정의어 (DDL)
----
```
* CREATE : 테이블 생성
* ALTER : 테이블 수정
* DROP : 테이블 삭제 (테이블 구조까지 아예 삭제됨, 롤백 불가)
* TRUNCATE : 테이블의 모든 행 삭제 (테이블 구조는 그대로, 데이터만 삭제)
```

데이터 조작어 (DML)
---
```
* SELECT : 데이터 조회
* INSERT : 데이터 삽입
* DELETE : 데이터 삭제 (롤백 가능)
* UPDATE : 데이터 수정
```
데이터 제어어 (DCL)
---
```
* GRANT : 권한을 부여
* REVOKE : 권한을 회수 
```
